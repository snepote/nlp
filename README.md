# nlp
Playing with Natural Language Processing and Sentiment Analysis

## Usage
### Syntax
```bash
$ bundle exec ruby syntax.rb "YOUR SENTENCE HERE"
```
### Sentiment
```bash
$ bundle exec ruby sentiment.rb "YOUR SENTENCE HERE"
```
